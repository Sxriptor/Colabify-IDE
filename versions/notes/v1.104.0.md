# Colabify IDE - 1.104.0

We are excited to announce the release of Colabify IDE version 1.104.0! This update focuses on enhancing the user experience, particularly for beginners, while also adding some visual improvements and usability enhancements to streamline your workflow. Below is a detailed overview of the new features and changes included in this release.

## New Features and Enhancements

### Improved UI for Beginners
- **Beginner-Friendly UI Enhancements**: We've introduced several UI changes aimed at making the Colabify IDE more intuitive for beginners. This includes cleaner layouts and more accessible navigation to help new users get started with ease.

### Visual Updates
- **Folder Icons**: To improve visual clarity and organization, we've added icons to folders. This addition makes it easier to differentiate between files and directories at a glance, enhancing the overall browsing experience.

### Enhanced Activity Bar
- **Collapsible Unused Items**: Unused items in the activity bar can now be collapsed into a single button. This feature declutters the interface by keeping only essential tools visible, allowing you to focus on your current tasks without distractions.

## New Welcome Experience

### Welcome Message Page
- **Introduction of a New Welcome Page**: A new welcome message page has been added to greet new users and provide them with the necessary resources to get started quickly. This page includes helpful tips, quick links to documentation, and guides for a smoother onboarding experience.

## Chat Panel Improvements

### Updated Chat Panel
- **Word Wrap Fix**: The chat panel has been updated to address previous glitches with text rendering. It now supports word wrapping, ensuring that long messages are displayed correctly without overlapping or truncation issues.
