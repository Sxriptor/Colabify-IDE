# Colabify IDE - 1.106.0

We are excited to announce the release of Colabify IDE version 1.106.0! This update brings several new features, enhancements, and critical fixes to improve your development experience. Below, you'll find a comprehensive breakdown of what's included in this release.

## New Features

### Notification System for Chat and Terminal
- **Enhanced User Interaction**: We have introduced a notification system for both the chat and terminal functionalities within Colabify IDE. Now, you will receive real-time notifications for important events and messages, ensuring you never miss an update while working on your projects.
- **Customizable Alerts**: Users can customize the type and frequency of notifications, allowing for a tailored experience that suits your workflow preferences.

## Beta Features

### Drag and Drop Implementation for Website and SaaS Building
- **Initial Rollout**: We are pleased to announce the initial implementation of drag and drop capabilities for website and SaaS building. This feature is currently in the beta stage, providing users with an intuitive and efficient way to design and organize components.
- **User-Friendly Interface**: The drag and drop feature aims to simplify the design process, enabling users to easily create and modify layouts without needing extensive coding knowledge.

## Bug Fixes

### Installation Process Improvements
- **Resolved Installation Issues**: We have addressed a critical issue where the IDE could become unresponsive during the installation of certain packages. This fix should ensure a smooth and hassle-free setup process for all users.
- **Stability Enhancements**: Additional improvements have been made to enhance overall stability and performance during the installation phase.

## General Enhancements

- **Performance Optimizations**: Various under-the-hood enhancements have been implemented to improve the speed and responsiveness of Colabify IDE.
- **User Interface Refinements**: Minor tweaks and adjustments have been made to the user interface for a more seamless and aesthetically pleasing user experience.

## Feedback and Support

We are continuously working to improve Colabify IDE and your feedback is invaluable to us. Please do not hesitate to reach out with any suggestions, issues, or feedback you may have. You can contact our support team or contribute to our community forums for additional assistance.

Thank you for choosing Colabify IDE. We hope you enjoy the new features and improvements in version 1.106.0!
